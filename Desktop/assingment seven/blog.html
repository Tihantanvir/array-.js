What is difference between map, filter, forEach and find ?

1. map() loop on every element in an array and returns a new array with the updated elements.

2. filter() check every element in an array with condition and return a new array that's element fulfil this condition.

3. forEach() loop on every element in an array but doesn't return anything, always forEach mathod return undefind.

4. find() check every element in an array with condition and return only first element.

what is Event loop ? How does work it ?

JavaScript is a single-threaded synchronous programming language. but sometime JavaScript can be aynchronus with event loop.

Event loop is just a guardian who keeps a good communication with Call Stack and Callback Queue. It checks if the call stack is free, then lets know the callback queue. Then Callback queue passes the callback function to Call stack to be executed. When all the callback functions are executed, the call stack is out and global execution context is free.
